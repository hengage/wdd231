Individual Project Site Plan

WDD 231: Web Frontend Development - German Basics Hub

1. Site Name

German Basics Hub

Reason for Site Name: The name was chosen because the website focuses on introducing the fundamentals of the German language in a simple, beginner-friendly way.

2. Site Purpose

The purpose of this website is to introduce beginners to learning German. The site will explain why learning German is useful, present basic vocabulary and phrases, and provide beginner-friendly tips and resources to help learners get started.

Target Audience

Beginners who are interested in learning German, especially students or self-learners with little or no prior knowledge of the language.

3. Scenarios (Audience Questions)

  1. How can I learn basic German words and phrases as a beginner?
  2. What resources or tips can help me practice German consistently?
